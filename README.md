# cognifyz_investment_analysis

## Overview
This project performs an end-to-end exploratory data analysis on an investment behavior dataset as part of the Cognifyz Data Analysis Internship.

## Tasks Covered
- Data Overview and Cleaning
- Gender Distribution Analysis
- Descriptive Statistics
- Most Preferred Investment Avenues
- Reasons for Investment
- Savings Objectives
- Information Sources
- Investment Duration Analysis
- Expectations from Investments
- Correlation Analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

## Key Insights
- Mutual Funds emerged as the most preferred investment avenue
- Long-term wealth creation is a primary savings objective
- Age shows a weak to moderate correlation with investment duration
